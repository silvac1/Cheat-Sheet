Html

Tags

Tags are everything in Html, without them there is no Html, tags are the elements that build up the webpage. For example, the < body > tag is used to create the body of a webpage.

Body

You use a body tag by creating a body element, like this:

< body >

< / body >

As you can see, the second tag has a forward slash ( / ) before the word body, this indicates that the specific tag is a closing tag, as you can guess, without the slash is called
an opening tag, you must have an opening tag and a closing tag to create an element. In this case, this is a body element, so I have an opening tag and a closing tag which creates
my body element.
